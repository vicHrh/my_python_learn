# 第一阶段练习说明

## 简介

这是为有Java背景的学习者准备的Python第一阶段练习题，涵盖了Python基础语法的核心内容。

## 练习内容

### 1. calculator_test.py
基础运算和函数定义练习：
- 实现基本数学运算（加减乘除）
- 数字分类函数
- 列表操作
- 字符串处理（回文检查）
- FizzBuzz游戏实现

### 2. data_types_test.py
Python数据类型操作练习：
- 列表操作（创建、增删改查、排序等）
- 字典操作（键值对管理、遍历等）
- 元组操作（创建、访问、拆包等）
- 集合操作（交集、并集、差集等）
- 字符串操作（格式化、分割、替换等）

### 3. control_structures_test.py
控制结构练习：
- 条件语句（if/elif/else）
- 循环语句（for/while）
- 嵌套循环（九九乘法表）
- 列表推导式
- 综合应用（猜数字游戏）

## 使用方法

1. 首先尝试独立完成每个练习中的TODO部分
2. 运行测试代码查看结果
3. 对照`solutions`目录中的参考答案进行对比学习
4. 修改和完善自己的实现

## 学习建议

1. **逐个完成**：建议按顺序逐一完成练习，确保掌握每个知识点
2. **动手实践**：不要只看代码，一定要亲自编写和运行
3. **对比差异**：思考Python和Java在实现相同功能时的语法差异
4. **查阅文档**：遇到不熟悉的函数或方法时，学会查阅Python官方文档

## 运行练习

```bash
# 运行计算器练习
python calculator_test.py

# 运行数据类型练习
python data_types_test.py

# 运行控制结构练习
python control_structures_test.py
```
